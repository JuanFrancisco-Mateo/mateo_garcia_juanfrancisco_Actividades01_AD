# Actividades de Acceso a Datos - DAM

Este repositorio contiene las actividades resueltas de la asignatura **Acceso a Datos** del ciclo de Desarrollo de Aplicaciones Multiplataforma (DAM) en el IES Carlos III de Cartagena. Cada ejercicio aborda distintos aspectos del tratamiento de archivos y carpetas utilizando **Java**.

## Índice de actividades

- **Actividad 1_1**: Lectura secuencial de archivos de texto (`FileReader`/`BufferedReader`). Filtrado de líneas y generación de sentencias SQL.
- **Actividad 1_2**: Procesamiento y filtrado de líneas en un archivo. Creación de carpetas, guardado de resultados y manejo de archivos.
- **Actividad 1_3**: Lectura de múltiples archivos, manejo de streams, representación y filtrado de datos por familias profesionales.

## Tecnologías y herramientas utilizadas

- **Java** (mínimo versión 17, recomendado 21 o 25)
- **Maven** como gestor de dependencias
- **Lombok** para simplificar código
- **IntelliJ IDEA** como entorno de desarrollo recomendado

## Cómo ejecutar las actividades

1. Clona el repositorio:

  - git clone https://github.com/JuanFrancisco-Mateo/mateo_garcia_juanfrancisco_Actividades01_AD.git

2. Abre el proyecto en **IntelliJ IDEA**.
3. Compila y ejecuta el paquete de la actividad que desees practicar.
4. Sigue las instrucciones del PDF adjunto para preparar los archivos de datos necesarios por cada ejercicio.
